# Personal Website

A static personal website built with React and Vite.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Run the development server:
   ```
   npm run dev
   ```

3. Build for production:
   ```
   npm run build
   ```

4. Run http-server:
    ```
    http-server dist -p 8080 -a 0.0.0.0
    ```
